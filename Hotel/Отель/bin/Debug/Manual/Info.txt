                		Инструкция

1. При первом запуске программы Вам будет предложено загрузить тестовый список клиентов. 
Если это не первый запуск, загрузится список клиентов из текущего файла.

2. Если Вы хотите использовать другой файл, то можете загрузить его. 

3. В любой момент Вы можете создать новый файл клиентов.
Для этого нужно будет указать имя создаваемого файла.

4.  Список клиентов автоматически сохраняется в текущем файле перед выходом из программы.

5. Для отказа от сохранения клиентов необходимо выйти по системной кнопке закрытия окна.
